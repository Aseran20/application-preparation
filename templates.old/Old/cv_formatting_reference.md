# CV Formatting Reference - Resume Adrian Turion

## Document Properties

### Page Setup
- **Format**: US Letter (8.5" × 11" / 216mm × 279mm)
- **Margins**:
  - Left: 0.32 inches
  - Right: 0.32 inches
  - Top: 0.55 inches
  - Bottom: 0.39 inches

### Document Structure
- **Layout**: Single table structure
- **Table dimensions**: 23 rows × 4 columns
- **Column widths**:
  - Column 1: 4.63 inches
  - Column 2: 4.63 inches
  - Column 3: 2.36 inches
  - Column 4: 0.80 inches

## Typography

### Font Family
- **Primary font**: Times New Roman (used throughout entire document)

### Font Sizes
- **22pt**: Name header (ADRIAN TURION)
- **10pt**: All other content (body text, titles, descriptions)

### Font Styles

#### Section Headers (INTRODUCTION, EDUCATION, WORK EXPERIENCE, etc.)
- Font: Times New Roman
- Size: 10pt
- Weight: **Bold**
- Decoration: **Underline**
- Color: Black

#### Company/University Names
- Font: Times New Roman
- Size: 10pt
- Weight: **Bold**
- Style: Regular
- Color: Black

#### Job Titles / Degree Names
- Font: Times New Roman
- Size: 10pt
- Weight: Regular
- Style: ***Italic***
- Color: Black

#### Location and Dates
- Font: Times New Roman
- Size: 10pt
- Weight: **Bold** (for locations)
- Style: ***Italic*** (for dates)
- Color: Black

#### Body Text / Descriptions
- Font: Times New Roman
- Size: 10pt
- Weight: Regular
- Style: Regular
- Color: Black

#### Subsection Headers (e.g., "Languages:")
- Font: Times New Roman
- Size: 10pt
- Weight: **Bold**
- Style: ***Italic***
- Color: Black

## Text Alignment
- **Primary alignment**: Left (default)
- **Right alignment**: Used for contact information, locations, and dates in header section

## Special Elements
- **Images**: 4 images embedded (profile photo)
- **Bullet points**: Not used (content organized in paragraphs)

## Color Palette
- **Text color**: Black only (no colors used)
- **No background colors**
- **No borders** (except table structure)

## Formatting Patterns by Section

### Header Section
```
ADRIAN TURION (22pt, Bold)
Address (10pt, Regular, Right-aligned)
Email | Nationalities (10pt, Regular)
Phone (10pt, Regular, Right-aligned)
```

### INTRODUCTION
```
INTRODUCTION (10pt, Bold, Underline)
[Summary paragraph] (10pt, Regular)
```

### EDUCATION
```
EDUCATION (10pt, Bold, Underline)

Institution Name (10pt, Bold)                  Location (10pt, Bold, Right-aligned)
Degree Name (10pt, Italic)                     Dates (10pt, Italic, Right-aligned)
• Bullet point details (10pt, Regular)
```

### WORK EXPERIENCE
```
WORK EXPERIENCE (10pt, Bold, Underline)

Company Name (10pt, Bold)                      Location (10pt, Bold, Right-aligned)
Job Title (10pt, Italic)                       Dates (10pt, Italic, Right-aligned)
• Responsibility/achievement (10pt, Regular)
```

### ADDITIONAL INFORMATION
```
ADDITIONAL INFORMATION (10pt, Bold)

Subsection: (10pt, Bold, Italic)
Details (10pt, Regular)
```

## Notes
- Consistent 10pt font size for all content except main name
- Heavy use of Bold and Italic for hierarchy rather than font size changes
- Underline used exclusively for major section headers
- Very tight margins to maximize content space
- Single-table layout for precise control over positioning
